# digiEduHack_hackathon
Main repository for the submission of the solutions for the DigiEduHack Hackathon.
Please read carefully all the documents provided.
To submit your solution, please:
- create your team branch.
- add any needed files/folders/dependencies to run your solution.
- fill the template shared to be submitted to the organization, DigiEduHack
- add your pdf/pptx in your branch for your pitch
